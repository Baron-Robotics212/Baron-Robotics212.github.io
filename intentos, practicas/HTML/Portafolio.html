Proyectos







Araña mecánica

Este proyecto trata de una araña en figura 3D, realizada a travéz de la aplicación de software Autodesk Fusión 360. Para sus movimientos se usaron la plataforma Arduino ID y el sistema de cómputo numérico MATLAB, con las placas Arduino nano y ESP32. En el video se puede observar como la araña saluda cuando se oprime un boton.

Alcancia digital

Este proyecto trata de una alcancía con figuras de Dragon Ball. Cuando se introducen monedas de $500 pesos o $1000 pesos, aparece en una pantalla LCD cuánto dinero se ha introducido y luego el total de dinero ahorrado. Para este proyecto se uso la plataforma Arduino ID a travéz de la placa ESP32.
